# 📌 Java 객체지향 (OOP)

## 객체지향이란?

**객체지향(Object-Oriented Programming)** 은 **객체를 중심으로 프로그램을 만드는 방식**이다.

 쉽게 말하면
**현실 세계처럼 프로그램을 만드는 것**

예시 💡

* 학생(Student)
* 자동차(Car)
* 게임 캐릭터

이런 것들을 **객체로 만들어서 관리**한다.

---

## 클래스와 객체 (비유)

 클래스 = **설계도 (붕어빵 틀)**
 객체 = **실제 결과물 (붕어빵)**

```java
class Student {
    String name;
    int age;
}
```

```java
Student s1 = new Student();
s1.name = "Kim";
s1.age = 17;
```

 하나의 설계도로 여러 객체를 만들 수 있다

---

## 객체지향의 특징 (비유 포함)

### 1️⃣ 캡슐화

 데이터를 하나로 묶기
 비유: **캡슐 안에 약이 들어있는 느낌**

---

### 2️⃣ 상속

 기존 클래스를 물려받기
 비유: **부모 → 자식 유전자 물려받기**

---

### 3️⃣ 다형성

 같은 이름인데 다르게 동작
 비유: **사람이 상황에 따라 다르게 행동**

---

### 4️⃣ 추상화

 필요한 것만 보여주기
 비유: **자동차 운전할 때 내부 엔진 몰라도 됨**

---

## 객체 사용 예시

```java id="k7d2qm"
class Student {
    String name;

    void sayHello() {
        System.out.println("Hello");
    }
}

public class Main {
    public static void main(String[] args) {

        Student s1 = new Student();
        s1.sayHello();

    }
}
```

---

## 정리

* 객체지향은 **현실 세계처럼 객체로 프로그램 구성**
* 클래스는 **설계도**, 객체는 **결과물**
* 캡슐화, 상속, 다형성, 추상화가 핵심 개념
